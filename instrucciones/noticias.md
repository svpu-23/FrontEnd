🧪 EVALUACIÓN:
🎯 ¿Qué vas a hacer?

Vas a crear una página web usando cajas (div), igual que el ejercicio que ya hiciste en clase.

🧱 PASO 1: CREAR EL HTML

Copia esto y trabaja sobre esto 👇

<body>

<div class="header">HEADER</div>
<div class="nav">NAV</div>

<div class="main">
    <div class="section">SECTION</div>
    <div class="aside">ASIDE</div>
</div>

<div class="article">ARTICLE</div>
<div class="footer">FOOTER</div>

</body>

👉 Esto es igual a lo que ya hiciste con .container y .item

🎨 PASO 2: CREAR EL CSS BASE


🧪 PASO 3: ORDENAR LAS CAJAS (IMPORTANTE)

Tu página debe verse así:

HEADER
NAV
[ SECTION | ASIDE ]
ARTICLE
FOOTER

👉 SECTION y ASIDE deben estar al lado (Flexbox)

🎯 PASO 4: DARLE ESTILO

Ahora mejora tu diseño:

✅ Cambia colores
✅ Agrega fondo (imagen o color)
✅ Agrega texto

Ejemplo:

.header{
    background-color: black;
    color: white;
}
✨ PASO 5: AGREGAR HOVER (OBLIGATORIO)

Debes usar al menos 1 hover (como el ejemplo que hicimos en clases)

🔵 PROYECTO 2: NOTICIAS
📰 ¿Qué debes hacer?

Crear una página de noticias usando cajas (div).
🧱 PARTE IMPORTANTE (LO NUEVO)
👉 Dentro de la caja SECTION, debes crear 3 cajas pequeñas
🪜 PASO A PASO (SIN CONFUSIÓN)

1️⃣ Ir a la SECTION
👉 Busca tu caja que dice:
SECTION

2️⃣ Crear 3 cajas dentro
👉 Dentro de esa caja debes tener:

Noticia 1
Noticia 2
Noticia 3

👉 Son como las cajas que hiciste antes (Uno, Dos, Tres)

3️⃣ Hacer que queden en fila (IMPORTANTE)
👉 A la SECTION debes ponerle:
display: flex
👉 Esto hace que:
Las noticias queden una al lado de la otra

4️⃣ Dar tamaño a cada noticia
👉 Cada noticia debe:
Tener tamaño (cuadrado o rectángulo)
Tener borde o color

5️⃣ Separarlas
👉 Debes dejar espacio entre ellas:
Puedes usar gap o margen
🎨 DISEÑO (OBLIGATORIO)

Cada noticia debe:

✅ Tener color distinto
✅ Tener texto (ej: “Noticia 1”)
✅ Tener borde visible

✨ HOVER (OBLIGATORIO)

Aplica hover en las noticias:

👉 Ejemplos:

Cambiar color
Agrandar
Sombra
🧠 TORPEDO (FÁCIL DE RECORDAR)

👉 “SECTION funciona como el container del ejercicio anterior”
👉 “Las noticias son como los item (cajas)”
